Transformación y Filtrado de Productos

Dado el array de productos ["Mesa", "Silla", "Lámpara", "Sofá", "Cama"]:

  1. Convierte cada producto a minúsculas.
  2. Filtra los productos que tengan más de 4 letras.
  3. Añade la palabra "producto" al inicio de cada elemento filtrado.
  4. Únelos en una sola cadena separada por comas.

Resultado esperado: "producto lámpara, producto silla"

Ejercicio 1: Procesamiento de Apellidos

Dada una lista de apellidos ["García", "López", "Martínez", "Sánchez", "Pérez"]:

  1. Filtra los apellidos que comienzan con la letra "M".
  2. Convierte los apellidos filtrados a minúsculas.
  3. Une los apellidos resultantes en una sola cadena separada por punto y coma.

Resultado esperado: "martínez"

Ejercicio 2: Análisis de Comentarios

Dado el array de comentarios ["Me gusta mucho este producto", "No me convenció", "Excelente calidad", "Podría ser mejor"]:

  1. Convierte todos los comentarios a minúsculas.
  2. Filtra los comentarios que contienen la palabra "me".
  3. Ordena los comentarios filtrados alfabéticamente.
  4. Únelos en una sola cadena separada por puntos y un espacio.

Resultado esperado: "me gusta mucho este producto. no me convenció. podría ser mejor"

Ejercicio 3: Construcción de Frases

Dado el array de palabras ["el", "perro", "corre", "rápido"]:

  1. Cambia la primera letra de cada palabra a mayúscula.
  2. Une las palabras en una sola cadena con espacios entre ellas.
  3. Añade un punto al final de la cadena para formar una oración.

Resultado esperado: "El Perro Corre Rápido."
