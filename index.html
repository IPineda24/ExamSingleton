Ejercicio: Implementación de un Adaptador para Convertir Pesos

En este ejercicio, trabajarás con dos clases: Pounds y Kilos. Cada una de estas clases maneja un peso en una unidad específica: libras o kilos, respectivamente.

La clase Pounds se utiliza para manejar pesos en libras, mientras que la clase Kilos se utiliza para manejar pesos en kilos. Cada clase tiene un método para obtener el peso en la unidad correspondiente:

  • getWeightInPounds(): Devuelve el peso en libras.
  • getWeightInKilos(): Devuelve el peso en kilos.

Tu tarea es implementar un Adaptador que permita convertir pesos de una unidad a otra, de manera que se puedan realizar conversiones entre libras y kilos fácilmente. Específicamente, debes crear una clase llamada WeightAdapter que:

  • Convierta el peso de kilos a libras cuando se llame al método getWeightInPounds().
  • Devuelva el peso en kilos cuando se llame al método getWeightInKilos().

Una vez que hayas implementado el adaptador, debes probar su funcionalidad realizando lo siguiente:

  1. Crear una instancia de la clase Pounds con un peso de 150 libras y mostrar el peso en libras.
  2. Crear una instancia de la clase Kilos con un peso de 68 kilos y luego utilizar el adaptador para convertir este peso a libras.
  3. Mostrar tanto el peso convertido a libras como el peso original en kilos utilizando el adaptador.

Recuerda que 1 kilogramo equivale aproximadamente a 2.20462 libras, y deberás usar esta conversión en tu adaptador.

¡Buena suerte y diviértete programando!

// Clase que maneja pesos en libras
class Pounds {
    constructor(weight) {
        this.weight = weight;
    }
 
    getWeightInPounds() {
        console.log("Peso en libras:", this.weight);
        return this.weight;
    }
}
 
// Clase que maneja pesos en kilos
class Kilos {
    constructor(weight) {
        this.weight = weight;
    }
 
    getWeightInKilos() {
        console.log("Peso en kilos:", this.weight);
        return this.weight;
    }
}

const weightInPounds = new Pounds(150);

// 2. Trabajar con peso en kilos a través del adaptador
const weightInKilos = new Kilos(68);
